## Commands

<!-- commands -->

- [`tsnl create [LIBNAME]`](#tsnl-create-libname)
- [`tsnl dev`](#tsnl-dev)
- [`tsnl build`](#tsnl-build)
- [`tsnl test`](#tsnl-test)
- [`tsnl lint`](#tsnl-lint)

### `tsnl create [LIBNAME]`

Init a library project.

### `tsnl dev`

Run the project in development mode.

### `tsnl build`

Build the project to dist for production.

### `tsnl lint`

Lint code with eslint.

### `tsnl test`

Run unitest with jest.
