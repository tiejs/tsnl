# Usage
<!-- usage -->
```sh-session
$ npm install -g tsnl
$ tsnl COMMAND
running command...
$ tsnl (-v|--version|version)
tsnl/0.0.5 darwin-x64 node-v10.16.3
$ tsnl --help [COMMAND]
USAGE
  $ tsnl COMMAND
...
```
